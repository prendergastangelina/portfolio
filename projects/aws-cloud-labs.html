AWS Cloud Labs

A collection of AWS Academy labs focused on cloud infrastructure, networking, storage, security, and monitoring.

Project Overview

I completed guided AWS labs that provided hands-on experience with creating, configuring, and managing cloud resources in a controlled learning environment.

Technologies Used

AWS EC2 Cloud Storage Virtual Networking Monitoring Cloud Security

Lab Topics

Lab topics included virtual machines, cloud service models, storage options, networking concepts, resource management, monitoring, security, and the responsibilities involved in operating cloud-based systems.

What I Learned

These labs helped me understand how cloud services are provisioned, connected, secured, and monitored, while reinforcing the differences between traditional infrastructure and cloud computing.
