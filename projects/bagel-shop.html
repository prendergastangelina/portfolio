Application Development

Bagel Shop Point-of-Sale System

An ASP.NET Web Forms application that processes customized customer orders and produces a calculated receipt.

Project Overview

I developed a point-of-sale system that allows a customer to enter their information, select bagels and add-ons, choose dine-in or takeout, and receive an itemized receipt.

Technologies Used

C# ASP.NET Web Forms HTML CSS

Key Features

Features include user input validation, conditional pricing, add-on selections, tax calculations, panel navigation, and receipt generation.

What I Learned

This project strengthened my understanding of event-driven programming, custom methods, return values, input validation, calculations, and ASP.NET server controls.
