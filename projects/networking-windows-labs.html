Networking & Windows Administration Labs

A collection of hands-on labs involving enterprise networking, Windows Server administration, identity management, and security.

Project Overview

Through a series of technical labs, I configured and managed Windows Server environments and practiced networking and systems administration tasks commonly used in enterprise IT.

Technologies Used

Windows Server 2022 Active Directory DNS Group Policy PowerShell TCP/IP

Lab Topics

Topics included Active Directory Domain Services, DNS configuration, Group Policy, DFS, BranchCache, remote access, VPNs, firewall configuration, monitoring, and endpoint security.

What I Learned

These labs strengthened my understanding of enterprise systems, identity and access management, network services, troubleshooting, and secure administration practices.
