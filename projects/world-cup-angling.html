Database Design

World Cup Angling Championship Database

A relational database designed to organize the participants, resources, and competition structure of an international angling event.

Project Overview

I designed this relational database while studying abroad in Ireland. The system was created to organize competition information involving entrants, boats, boat personnel, tournament sections, and statuses.

Technologies Used

SQL RDBMS ER Modeling Normalization Database Design

Database Design

I identified the required entities, attributes, primary keys, foreign keys, and relationships needed to accurately represent the competition structure while reducing redundant data.

What I Learned

This project strengthened my ability to translate business rules into a structured relational database and create an entity-relationship model that supports data integrity.
